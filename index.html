📄 Quick Reference

Use while taking the test

Prompt Formula

Every strong prompt includes:

Deliverable+ Inputs+ Specifics+ Guidance

Prompt Techniques

① Be Clear & Direct
Start with an action verb. State the task immediately.
✕ "I was thinking about geothermal energy…"
✓ "List 3 countries using geothermal energy."
② Be Specific
Add constraints: length, format, audience, tone.
✕ "Write a short story about a character."
✓ "Write a story under 1,000 words with one supporting character."
③ Provide Example (Few-shot)
Show Claude input → output pairs before your real request.
Zero-shot — no example, direct instruction
Few-shot — 1–3 examples first, then your task

4D Framework

Discernment

Evaluate AI output critically before using it.

Description

Communicate clearly. Precise prompts get precise results.

Delegation

Decide what to delegate to AI and what to keep yourself.

Diligence

Interact responsibly. Responsibility stays with you.

Chat vs Cowork

SituationUse
Quick Q&A, brainstorming, single-turn draftsChat
Task needs to read/edit files on your computerCowork
Multi-step task spanning multiple toolsCowork
Task should repeat on a scheduleCowork
Project needing persistent context across sessionsCowork

Key Concepts

ProjectScoped workspace with instructions, scheduled tasks, context, and memory.
SkillExpertise package that teaches Claude a specific task type. Auto-activates when added.
PluginA packaged set of skills built around a job function.
ConnectorDriver that lets Claude connect to external apps (e.g. Google Drive, Slack).
ArtifactStandalone output created alongside your chat — doc, code, SVG, etc.
Global Inst.Brief that applies to every session: your role, shorthand, output preferences.
ScheduledRecurring task inside a Project. Always test on drafts first.

Safe vs Unsafe Inputs

✓ Safe

  • Public documents
  • Non-sensitive meeting notes
  • Your own drafts
  • Anonymised data
  • Internal process guides

✕ Never share

  • Customer PII
  • Passwords / API keys
  • Unpublished financials
  • Trade secrets / IP
  • HR / health records
Cowork safety: Always back up files before running new workflows. Test on copies first. Avoid vague destructive verbs like "cut", "delete", or "update".
